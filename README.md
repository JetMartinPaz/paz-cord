Este es un proyecto personal - PW para chats.

## Comenzamos

Dev Server:

```bash
npm run dev
```

Prisma
```bash
npx prisma studio

// cambios
npx prisma generate
npx prisma db push
```

Abrir [http://localhost:3000](http://localhost:3000)
