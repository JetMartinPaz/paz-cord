Chat - tipo Discord con Next.js, React, Prisma, Tailwind, MySQ.

## Comenzamos

Dev Server:

```bash
npm i
npm run dev
```

Prisma
```bash
npx prisma studio

// cambios
npx prisma generate
npx prisma db push
```

Abrir [http://localhost:3000](http://localhost:3000)
